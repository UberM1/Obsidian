```bash
pacman -S <package>   # instala un package
pacman -Sy <package>  # instala un paquete y actualiza los sources
pacman -Syu <package> # actualiza todos los sources y hace un update de los                          paquetes
pacman -Si <package>  # Da info del paquete, lugar que se instala, version,                          descripcion, source, si tenes mas de un repo con                             esto podes ver la diferencia entre los distintos                             sources.
```
